# 서비스 이용약관

최종 갱신: 2026-05-25

> 본 약관은 haru (이하 "운영자") 가 제공하는 보이스 채팅 기반 크로스언어 매칭 서비스(이하 "서비스") 의 이용 조건을 규정합니다.

## 제1조 (목적)

본 약관은 운영자가 제공하는 서비스의 이용 조건 및 절차, 운영자와 회원의 권리·의무 및 책임 사항을 규정함을 목적으로 합니다.

## 제2조 (정의)

본 약관에서 사용하는 용어의 정의는 다음과 같습니다.

1. "운영자" — haru 서비스를 운영·제공하는 자. 운영자 표시 정보는 본 약관 하단 부칙에 게시합니다.
2. "회원" — 본 약관에 동의하고 서비스 이용 계약을 체결한 자.
3. "보이스 클론" — 회원이 등록한 음성 샘플로부터 ElevenLabs 의 음성 합성 모델이 생성한 회원 고유의 음성 ID 및 합성 결과물.
4. "메시지 음성" — 회원이 채팅에서 전송한 텍스트가 발신자의 보이스 클론으로 합성·번역되어 수신자에게 재생되는 음성 파일.
5. "보이스 한마디" — 회원이 프로필에 등록하는 짧은 자기소개 텍스트와 이를 보이스 클론으로 합성한 음성. 매칭 탐색 단계에서 다른 회원에게 노출됩니다.
6. "콘텐츠" — 회원이 서비스 내에서 작성·전송·게시하는 모든 텍스트, 음성, 사진을 통칭합니다.

## 제3조 (약관의 게시와 개정)

1. 운영자는 본 약관의 내용을 회원이 쉽게 알 수 있도록 서비스 초기 화면 및 본 페이지에 게시합니다.
2. 운영자는 관련 법령에 위배되지 않는 범위에서 본 약관을 개정할 수 있으며, 개정 시 적용 일자 및 개정 사유를 명시하여 시행일로부터 최소 7 일 전 (회원에게 불리한 개정의 경우 최소 30 일 전) 부터 시행일까지 공지합니다.

## 제4조 (회원가입 및 본인 확인)

1. 서비스 이용자는 만 18 세 이상이어야 가입할 수 있습니다.
2. 회원은 가입 시 운영자가 요구하는 정보 (이메일, 생년월일, 성별, 국적, 사용 언어, 프로필 사진, 보이스 클론용 음성 샘플) 를 제공해야 합니다.
3. 운영자는 신고 누적 또는 미성년자 의심 사유로 가입 거부 및 계정 정지 조치를 취할 수 있습니다.

## 제5조 (서비스의 제공)

1. 운영자는 회원에게 다음 서비스를 제공합니다.
   - 매칭 탐색 및 좋아요·매치
   - 매칭 상대와의 채팅 (텍스트, 자동 번역, 발신자 보이스 클론 음성 합성)
   - 보이스 한마디 등록 및 자동 번역·합성
2. 서비스의 일부는 외부 서비스 (ElevenLabs, OpenAI, Google Vertex AI, Supabase) 를 사용합니다. 자세한 처리 위탁 내역은 개인정보처리방침 제3조 참조.

## 제6조 (회원의 의무)

회원은 다음 행위를 해서는 안 됩니다.

1. 타인을 사칭하거나 허위 정보를 등록하는 행위.
2. 미성년자가 가입하거나 미성년자임을 알고도 접근을 시도하는 행위.
3. 성적, 마약 관련, 자해 권유 등 본 약관 제7조에서 정한 차단 카테고리에 해당하는 콘텐츠를 작성·전송하는 행위.
4. 타인의 음성·사진·이미지를 무단 사용하거나 보이스 클론을 사칭·도용 목적으로 생성하는 행위.
5. 외부 채널 (메신저 ID, 전화번호, 이메일 등) 로 유도하는 스팸·광고 행위.
6. 서비스의 안전성을 저해하는 자동화된 접근, 크롤링, 역공학, 어뷰징 행위.

## 제7조 (콘텐츠 모더레이션)

본 조는 모든 메시지 및 보이스 한마디에 적용되며, 카테고리 정의 및 차단 방식은 다음과 같습니다.

**(1) 차단 카테고리**

다음 4 개 카테고리에 해당하는 표현은 자동 차단됩니다.

- 성적 콘텐츠 (sexual)
- 마약 관련 콘텐츠 (drug)
- 미성년자 대상·관련 표현 (minor)
- 자해 권유의 타인 명령형 (self_harm — 한국 형법 §252, 일본 刑法 §202 자살관여죄 형사처벌 정합)

(harassment / hate / violence 카테고리는 1차 출시 차단 대상에서 제외하며 운영 데이터 누적 후 결정합니다.)

(이용자 본인의 1인칭 위기 신호 — 예: "죽고 싶어" — 는 차단하지 않으며, helpline 연계 분기는 후속 출시 예정.)

**(1-1) 검증 절차 — 사전 키워드 차단 + OpenAI 2차 검수**

운영자는 다음 2 단계로 콘텐츠를 검증합니다.

1. **사전 키워드 차단** — 운영자가 운영하는 정규화된 키워드 사전 (한국어, 일본어, 영어) 으로 메시지 및 보이스 한마디를 검사. 키워드 매칭 시 즉시 차단.
2. **OpenAI Moderation 2차 검수** — 사전 차단을 통과한 텍스트를 OpenAI Inc. (미국 캘리포니아 소재) 의 Moderation API (`omni-moderation-latest`) 로 송신해 2차 검사. OpenAI 의 처리방침이 적용되며, 운영자는 별도로 저장하지 않습니다. OpenAI 의 처리 정책은 https://openai.com/policies 참조.

차단 응답 본문은 카테고리·매칭 키워드를 노출하지 않습니다 (우회 패턴 학습 방지).

**(2) 자동 정지 (누적 신고)**

같은 회원에 대한 신고가 일정 수 (출시 시점 기준 3 건) 누적되면 운영자는 사전 통지 없이 해당 계정을 자동으로 정지할 수 있습니다. 정지된 회원은 앱 내 안내 모달을 통해 정지 사실 및 이의제기 채널 (CS 이메일) 을 확인할 수 있습니다.

**(3) audit 기록 보존 (1년)**

다음 audit 이력은 발생 시점부터 1 년간 보존된 뒤 자동 폐기됩니다.

- 콘텐츠 차단 이력 (sender_id, 카테고리, 언어, 시각)
- 자동 정지 발동 이력 (frozen_user_id, 신고자 ID 배열, 발동 시각)
- 사용자 신고 이력 (reporter_id, reported_id, 사유, 시각)
- 사용자 차단 이력 (blocker_id, blocked_id, 시각)

audit 이력은 운영 목적 (재범자 식별, 신고 패턴 분석, 정지 처분 및 분쟁 협조의 근거 보존) 으로 보존되며, 회원 탈퇴 후에도 동일하게 적용됩니다. 자세한 보존 근거 및 회원 탈퇴 시 처리에 관해서는 개인정보처리방침 제8조를 참조하십시오. 메시지 원문 / 음성 파일 / 사진 / 매칭 토큰은 audit 에 포함되지 않습니다.

## 제8조 (음성 데이터의 처리)

본 조는 회원이 등록한 보이스 클론 학습 음성, 메시지 음성, 보이스 한마디 음성 모두에 적용됩니다. 자세한 항목은 개인정보처리방침 제4조를 함께 참조하십시오.

**(1) 보이스 클론 학습 음성**

회원이 보이스 클론 생성을 위해 등록한 음성 샘플은 ElevenLabs 가 음성 ID 를 생성한 즉시 운영자 인프라에서 폐기됩니다. 원본 학습 음성은 운영자가 보관하지 않습니다.

**(2) ElevenLabs 음성 ID**

ElevenLabs 가 보유하는 회원의 음성 ID 는 회원이 보이스 클론을 재녹음할 때까지 유지되며, 재녹음 시 이전 음성 ID 는 ElevenLabs API 를 통해 삭제 요청됩니다. 회원 탈퇴 시에도 운영자가 ElevenLabs API 로 삭제를 요청합니다.

회원은 보이스 클론 단독 삭제를 별도로 요청할 수 없으며, 재녹음(덮어쓰기) 또는 계정 탈퇴를 통해서만 삭제할 수 있습니다.

**(3) 메시지 음성 (30 일 자동 폐기)**

채팅에서 합성된 메시지 음성 파일은 **수신자가 청취한 시점으로부터 30 일이 경과** 하면 운영자의 Storage 에서 자동으로 폐기됩니다. 텍스트 및 자동 번역 결과는 매치 생명주기와 묶여 유지됩니다. 회원이 폐기된 음성을 다시 듣고 싶을 때는 ElevenLabs 재합성을 요청할 수 있으며, 이 경우 회원의 현재 보이스 클론으로 새로 합성됩니다 (재녹음으로 음성 ID 가 변경된 경우 새 음성이 합성될 수 있습니다).

미청취 메시지 음성은 30 일 경과 이후에도 자동 폐기 대상이 아닙니다 (수신자가 청취할 때까지 보존).

송신자가 계정을 탈퇴한 경우에도 송신자가 수신자에게 보낸 미청취 메시지 음성은 수신자의 청취 권리 보호를 위해 매치가 유지되는 동안 잔존하며, 청취 완료 후 30 일 자동 폐기 정책의 적용을 받아 송신자 탈퇴 후에도 최대 30 일까지만 잔존한 뒤 자동 폐기됩니다.

**(4) 보이스 한마디 음성**

보이스 한마디 음성은 회원이 텍스트를 변경하거나 계정을 탈퇴할 때 폐기됩니다.

## 제9조 (회원 탈퇴 및 데이터 삭제)

1. 회원은 언제든지 서비스 내 [설정 → 계정 → 회원 탈퇴] 경로로 탈퇴를 요청할 수 있습니다. 앱 외 채널은 CS 이메일 (sejinim02@gmail.com) 로 가입 이메일을 명시한 삭제 요청을 발송할 수 있습니다.
2. 탈퇴 시 회원의 데이터는 다음 4 카테고리로 처리됩니다.
   - **즉시 삭제** — 프로필 사진, 보이스 한마디 음성, 매칭 선호도, 디바이스 푸시 토큰, 매칭 스와이프 이력, 매치 뮤트 이력은 운영자 인프라에서 제거됩니다. ElevenLabs 음성 ID 는 외부 API 삭제 요청이 발송됩니다 (제8조 (2)). 사용자 신고·차단 이력은 분쟁 협조 목적으로 1 년 보존 후 자동 폐기됩니다 (제7조 (3) 및 개인정보처리방침 제8조).
   - **즉시 익명화** — 이메일 / 비밀번호 / 표시명 등 식별 정보는 즉시 익명화되며 (`deleted-{UUID}@deleted.local`) 회원은 더 이상 로그인할 수 없습니다. 매치 화면에서는 "탈퇴한 사용자" 로 표시됩니다.
   - **1 년 보존 후 자동 폐기** — 콘텐츠 차단·자동 정지·신고·차단 audit 이력은 분쟁·수사 협조 목적으로 1 년 보존 후 자동 폐기됩니다 (제7조 (3) + 개인정보처리방침 제8조).
   - **영구 보존** — 매치 / 메시지 본문 / 자동 번역 결과는 상대 회원의 채팅 기록 보호를 위해 매치 단위로 영구 보존됩니다. 메시지 발신자 식별 정보는 익명화되므로 상대 회원은 "탈퇴한 사용자" 의 메시지로 인식합니다.
3. 단, 관련 법령상 별도의 보존 의무가 있는 정보는 해당 기간 동안 별도 분리 보관 후 파기합니다.

## 제10조 (분쟁 해결을 위한 정보 보존)

운영자는 회원 간 분쟁 해결, 수사기관의 자료요청, 정지 처분에 대한 이의제기 대응을 위해 제7조 (3) 에서 정한 audit 이력을 회원 탈퇴 후에도 발생 시점부터 1 년간 보존합니다. 본 보존은 개인정보 보호법 제21조 제1항 단서 (다른 법령에 따른 보존 예외) 및 같은 조 제3항 (분리 저장·관리), 정보통신망법 제44조의10 (명예훼손 분쟁조정부), 전기통신사업법 제83조 (수사기관의 통신자료 제공 요청) 에 따른 처리이며, 자세한 사항은 개인정보처리방침 제8조를 참조하십시오.

## 제11조 (책임의 제한)

1. 운영자는 천재지변, 외부 서비스 (ElevenLabs, OpenAI, Vertex AI, Supabase) 의 장애, 회원의 귀책 사유로 인한 손해에 대해 책임을 지지 않습니다.
2. 운영자는 회원 간의 분쟁에 직접 개입하지 않으며, 다만 콘텐츠 모더레이션 정책 (제7조) 에 따라 차단·정지 조치를 취할 수 있습니다.

## 제12조 (준거법 및 관할)

본 약관은 대한민국 법에 따라 해석되며, 분쟁 시 운영자의 주소지 관할 법원을 1심 관할 법원으로 합니다.
